shallow copy of nvar elements in dimension dim of an array, starting at index=indx. Copies pointer to data, but not data itself.

Shallow slice is only able to address the "nvar" range in "dim", and all entries of the src array for d<dim (cannot access any nx4=2, etc. entries if dim=3 for example)
